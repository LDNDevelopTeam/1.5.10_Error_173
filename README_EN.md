## How to Run

### Client
**Requirements:** Java 8 (Zulu recommended)  
[Download Zulu Java 8](https://www.azul.com/downloads/?version=java-8-lts&os=windows&architecture=x86-64-bit&package=jdk#zulu)  

1. Download the client zip file:  
[Error_173.zip](https://github.com/LDNDevelopTeam/1.5.10_Error_173/releases/latest)

2. Install **MultiMC** or **PrismLauncher**:  
- [MultiMC](https://multimc.org/#Download)  
- [PrismLauncher](https://prismlauncher.org)  

   After installing, log in with your Minecraft account and **drag & drop** the downloaded zip file.  

3. After installation, set the Java path to the Zulu Java 8 you installed earlier. Adjust memory allocation if needed, then click launch.  

4. Start playing!
